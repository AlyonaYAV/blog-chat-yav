## "Funy Pets" is a blog for those who have cute pets.

Do you have a pet? If not, you should have it!
Because I wrote this app for you:-). Here you can chat with other pet owners.

The stack of used technologies:

* Nuxt.js
* Vuex
* Vue.js
* Express
* Socket.IO
* MongoDB
* Vuetify
* Axios
* (PWA)
* Element UI
